# Stationery Inventory Management System

A Python program that manages a stationery shop's inventory. It allows users to add, edit, update, display, and export stationery items.

## Features

- **Add new items** - Enter new stationery items with name, quantity, and cost
- **Edit items** - Update the details of existing items
- **Record sales** - Deduct sold quantities from stock
- **Display inventory** - View all items in a formatted table
- **Export to CSV** - Save the inventory to a `.csv` file

## Files

| File | Description |
|------|-------------|
| `main.py` | Main program with menu system and inventory logic |
| `stationeryclass.py` | Contains the `Stationery` class used to calculate total cost |

## How to Run

```bash
python main.py
```

## Requirements

- Python 3.x
